## 📌 프로젝트 소개
이 프로젝트는 Django 기반의 백엔드 포트폴리오입니다.   
여러 페이지를 만들고 만들고자 하는 기능을 구현하고자 합니다.

## 🚀 주요 기능 (계속 추가 예정)
- Django 최신 버전 기반 프로젝트 구성
- RESTful API 개발
- PostgreSQL 데이터베이스 연동
- AWS 배포 (EC2, S3, RDS 등)
- JWT 인증 및 사용자 관리
- ⏳ GraphQL 적용
- ⏳ CI/CD 구축
